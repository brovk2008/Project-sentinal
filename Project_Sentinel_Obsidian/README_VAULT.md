# Project Sentinel — Obsidian Knowledge Vault

Welcome to the Project Sentinel Knowledge Vault. This vault contains comprehensive technical documentation, architecture diagrams, and operational intelligence for Project Sentinel — a hackathon-grade, production-quality crime intelligence platform built on real Karnataka Police FIR data.

## How to Use This Vault

1. Open Obsidian → **Open folder as vault** → Select `Project_Sentinel_Obsidian/`
2. Enable **Graph View** to see document interconnections
3. Enable **Dataview** plugin for interactive tables
4. Use `Ctrl+O` to navigate between notes
