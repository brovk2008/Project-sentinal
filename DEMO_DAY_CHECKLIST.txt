PROJECT SENTINEL — DEMO DAY READY CHECKLIST
===========================================

[ ] PHYSICAL & SYSTEM STATE
    - Power cable connected (never demo on battery).
    - Windows Sleep settings: Screen turn-off and sleep set to 'Never'.
    - Network: Local Wi-Fi stable OR Cellular Hotspot active as a backup.
    - Sound & Notifications: 'Do Not Disturb' mode activated on Windows.
    - Browser: All unrelated tabs, folders, and personal windows closed.

[ ] INFRASTRUCTURE BOOT SEQUENCE

    - Confirm Docker Desktop launches and "sentinel_postgres" container starts (Up).

    - Confirm Python FastAPI starts in new minimized window.
    - Confirm Cloudflare Tunnel establishes and public URL copies to clipboard.

[ ] AI & RAG PRE-WARMUP VERIFICATION

      * "Warm Cache: Network statistics cache warmed."

[ ] FRONTEND CONFIGURATION

    - Open the frontend client at http://localhost:5173 or using the public tunnel.
    - Perform a manual "Hard Refresh" (Ctrl + F5) to clear cache and reload files.

[ ] FLOW REHEARSAL & BACKUPS
